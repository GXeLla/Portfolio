## 2026-05-05 — Active Navigation Indicator

### Concept

A navigation system where the active link is controlled by click interaction. The active state is visually emphasized with a minimal left-side indicator and subtle hover glow effects for better UI feedback.

### Behavior

- Clicking a link sets it as the **active page**
- Only one link is active at a time
- Active link displays a **2px vertical bar on the left side**
- Active link has a subtle background highlight
- Hovering the active link creates a soft “light spread” effect
- Non-active links remain clean and minimal

### Visual Style

- Left indicator: 2px solid white or soft gradient line
- Active background: low-opacity white overlay
- Hover effect:
  - Soft glow spreading from the left edge
  - Fade-out transparency using gradient (linear or radial)

### Optional Enhancements

- Smooth sliding animation when switching active link
- Add slight easing (cubic-bezier) for premium feel

### Concept

Buttons simulate a physical press feeling when clicked, adding weight and realism to UI interactions.


### Behavior

### Visual Style
